# 💰 Personal Finance Tracker

A Full Stack Personal Finance Tracker built using **Spring Boot**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- 👤 User Management
- 💳 Multiple Accounts
- 💰 Income & Expense Tracking
- 📊 Budget Management
- 🎯 Financial Goals
- 🔄 Recurring Transactions
- 📈 Dashboard Analytics
- 🔐 JWT Authentication (In Progress)

---

## 🛠 Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
